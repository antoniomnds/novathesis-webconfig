Welcome to the NOVAthesis template configuration!
Below and in the following pages you'll have the possibility to set document preferences that will have repercussions on the .tex file and thus in the presentation of your future academic work.
In the end, you can generate this file and replace it with the original one in the /LaTeX folder of the NOVAthesis template. As such, it is recommended (and necessary) that you first download the latest release, available at the NOVAthesis releases page.

Document cover selection
